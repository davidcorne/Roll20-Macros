# General Macros

## Secret Roll

A nice basic macro for secret rolls

```
/w gm &{template:default} {{name=Group Secret Check}} {{Player 1=[[1d20]]}} {{Player 2=[[1d20]]}} {{Player 3=[[1d20]]}} {{Player 4=[[1d20]]}} {{Player 5=[[1d20]]}} {{Player 6=[[1d20]]}}
```
